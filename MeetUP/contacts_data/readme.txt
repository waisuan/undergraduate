You can import the contacts.vcf file into your emulator by following the steps below:

1. Run your emulator through the Android Development Tools Plugin in Eclipse

2. Open the DDMS Perspective and select your emulator under the Devices view.

3. Go to the file explorer view and select the /mnt/sdcard folder.

4. Select the "push a file onto device" button on the top right corner of the File Explorer View and select the contacts.vcf file

5. For Android 2.3.3 go to the contacts application in your emulator and press the menu button, For Android 4.2 go to people application

6. Select the Import/Export menu and then select "Import From SD Card" option. The contacts application will automatically locate the contacts.vcf file in your emulator and import the list.
